## 🇹🇷 TÜRKÇE

### Projenin Amacı

Bu araç, tasavvuf ve İslam ezoterizmindeki kadim sayısal sistemleri — başta ebced ilmi olmak üzere — modern web teknolojisiyle buluşturmak amacıyla geliştirilmiştir. Tek dosyada çalışan, kurulum gerektirmeyen, tamamen tarayıcı tabanlı bir araçtır.

Amaç; bu ilimlerle fiilen meşgul olan talebelerin, araştırmacıların ve yol ehli kişilerin hesaplama süreçlerini kolaylaştırmaktır. Kesin sonuç üretmez, hüküm vermez — yalnızca **olası ilişkileri ve bağlantıları görünür kılar**. Sonuçları yorumlamak kullanıcıya aittir.

> **Fikir ve vizyon:** Şahsıma aittir.  
> **Kodlama ve tasarım:** Claude (Anthropic) ile birlikte geliştirilmiştir.  
> **Algoritma desteği:** ChatGPT katkı sağlamıştır.

## 🇬🇧 ENGLISH

### Purpose of the Project

This tool was developed to bridge the ancient numerical systems of Sufism and Islamic esotericism — primarily the science of Abjad — with modern web technology. It is a single-file, browser-based application that requires no installation.

The goal is to assist students, researchers, and those genuinely on the spiritual path in their work within these disciplines. It does not produce definitive results or pass judgments — it only **makes possible connections and relationships visible**. The interpretation belongs to the user.

> **Vision & concept:** The author.  
> **Coding & design:** Developed alongside Claude (Anthropic).  
> **Algorithm support:** ChatGPT contributed.

---

### Contents and Usage Examples

| Module | Description | Example |
|---|---|---|
| **Abjad Calculator** | Arabic text → abjad value | `بسم الله` → 786 |
| **Abjad → Letter** | Generates Arabic letter combinations from a number | `786` → possible letter strings |
| **Color ↔ Abjad** | Converts abjad value to HEX/RGB color | `الله` (66) → #420000 |
| **Abjad → Sound** | Converts value to frequency and plays tone | `66` → 66 Hz tone |
| **Abjad → Location** | Maps value to latitude/longitude | Symbolic geographic analysis |
| **Character Convert** | Conversion between Arabic script forms | Abjad notation styles |
| **Letter → Element** | Associates each letter with one of four elements | `ن` → Water |
| **Asma Finder** | Finds Divine Names matching a given abjad value | `99` → related names |
| **Vefk (Magic Square)** | Generates numerical square matrices | 3×3, 4×4 vefk |
| **Day / Hour** | Planetary hours and days table | Which planet rules which hour |
| **Verse by Name** | Shows Quranic verses close to a name's abjad value | Research reference |
| **Compatibility** | Calculates harmony between two names | Traditional method |
| **Personal Number** | Personal numerical analysis | Birth date, name |
| **Dhikr Count** | Dhikr count and abjad relationship | `33`, `99`, `786` |
| **Quran** | Surah/verse search and analysis | Text research |
| **Frequency Analysis** | Letter frequency within a text | Prayer text analysis |

### ⚠️ Warnings Against Misunderstanding

- This tool is **not** a divination, fortune-telling, or magic instrument.
- Calculation results are **not definitive truths**; they offer a reference and ground for reflection.
- The science of Abjad is an **interpretive discipline**; results support inquiry, not decision-making.
- Those seeking to "open fortune" or "break bindings" are **not the target audience** of this tool.
- This tool does **not represent any tariqa, community, or organization**.
- Results cannot be used as a basis for medical, legal, or spiritual **decision-making**.

---

### ✅ Who Should Use It?

- Students who are **actively on the Sufi path**
- Researchers engaged with Abjad, Asma, and Vefk at an **academic level**
- Theologians studying Islamic esotericism from an **academic or cultural** perspective
- Individuals working **under the guidance** of a murshid or master
- Sincere beginners approaching these sciences with **genuine respect, courtesy, and inquiry**

---

### 🚫 Who Should Stay Away?

- Those approaching with **surface-level curiosity** or for entertainment
- Those wishing to practice deeply **without a spiritual guide**
- Those who wish to treat results as **absolute truths or binding answers**
- Those wishing to use these sciences for **commercial or manipulative** purposes
- Those who wish to use the tools as a basis for belief or spiritual **decision-making**
